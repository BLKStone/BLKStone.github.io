docker容器快速部署Nessus（Linux篇）
地址

安装docker

装载系统

docker pull NAME 即可pull到本地，完成后启动即可(具体可以参考上一篇文章《在docker容器中运行kali linux （windows版）》)
为了介绍参数的使用和避免跨洋数据传输产生的网络不稳定，我已经
通过 docker save 【IMAGE ID】 -o nessus.tar 下载

导入步骤

完成后可以通过docker images 查看，如果你的和我一样显示为 可以执行下面的命令
root@vamtest:~#docker tag e714228e7aac nessus:latest 其中， e714228e7aac为IMAGE ID可以通过docker images查看，如下图，执行后再次输入docker images发现已经成功修改

启动

直接访问https://本机ip:8834 ， 如图

获取免费激活码
地址

如过需要进入容器，执行
root@vamtest:~#docker exec -it 568e84c835a0 bash 把568e84c835a0替换为你的CONTAINER ID即可，可以通过docker ps获得

停止容器

root@vamtest:~#docker stop 【CONTAINER ID】 //CONTAINER ID，可以通过docker ps获得
