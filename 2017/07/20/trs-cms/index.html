简介

TRS

常见产品

平台级产品

TRS内容协作平台(TRS WCM)
涵盖网站建设、内容服务、内容传递等内容价值链的各个方面。具有元数据管理、自主建站、大规模网站群应用、场景式服务、多媒体管理等功能。

TRS文本挖掘软件（TRS CKM）
文本挖掘是一种数据挖掘技术，能够实现对文本内容的智能分析。TRS CKM是中文文本挖掘应用的开发接口，包括自动分类、自动聚类、自动排重、信息抽取、关联关系挖掘等，是中国第一个真正实用化的文本挖掘产品。

TRS身份服务器系统（TRS IDC）
实现各种应用系统间跨域的单点登录和统一的身份管理功能。提供与第三方协作应用系统集成的框架以及非常便捷的二次开发接口。

应用级产品

TRS网络信息雷达系统（TRS InfoRadar）
通过聚焦定制采集源，进行实时监控，把最新网页采集到本地，并通过文本智能技术进行内容处理，保证网络信息采集实时性和全面性。

TRS全文数据库系统（TRS Database Server）
实现对结构化数据和非结构化数据的统一管理和全文检索，同时，采用分布式集群架构，满足用户海量数据和高并发环境下的检索性能和可靠性要求。

TRS互联网舆情管理系统（TRS OM）
通过互联网信息采集和文本挖掘技术，帮助各级政府快速发现和收集所需的社会网络舆情信息，通过自动采集、自动分类、智能过滤、自动聚类、主题检测和统计分析，实现社会热点话题、突发事件、重大案情的快速识别和定向追踪，从而帮助政府及时掌握舆情动向，对有较大影响的重要事件快速发现、快速处理，从正面引导舆论和宣传，构建积极向上的主流舆论，并为政府决策提供信息依据。
